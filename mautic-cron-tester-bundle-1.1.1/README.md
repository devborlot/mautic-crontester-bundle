# Cron Tester Bundle for Mautic 3

Download link for Mautic 2 is included in the package in file mautic2-version.md.

Run commands directly from the Mautic and speed up your tests.                       

Mautic requires a few cron jobs to handle some maintenance tasks.  
This bundle add new buttons execute single cron job directly from Mautic.
We highly recommend use it just for **test purpose**.

**Segment rebuild**  
**Campaign rebuild**  
**Campaign trigger**

### Installation

1. Unzip files to plugins/MauticCronTesterBundle
2. Clear cache (app/cache/prod/)
3. Go to plugins and and click to button Install/Update Plugins
